🌦️ Weather API App

A simple Weather Application built using HTML, CSS, and JavaScript (Fetch API + Async/Await). Users can enter any city name to get real-time weather information using the OpenWeatherMap API.

🚀 Features

🌍 Search weather by city name

🌡️ Current Temperature

💦 Humidity

🌬️ Wind Speed

☁️ Weather Condition

❌ Invalid city error handling

📱 Responsive UI

⚡ Uses Fetch API with Async/Await


🛠️ Technologies Used

HTML5

CSS3

JavaScript (ES6)

Fetch API

Async / Await

OpenWeatherMap API
